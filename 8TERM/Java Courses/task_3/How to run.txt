1. Открыть проект "JavaService" в IntelliJ IDEA
2. Запустить класс "Charts", который находится в JavaService/src/main/java/hello/Charts
3. Откроектся окно, в котором мы можем просмотреть графики:
	Вам будет показана информация по курсам валют за 2015, 2016, 2017 года и график изменений курса по каждой валюте.
4. Чтобы получить информация по определенной валюте за все 3 года, следует выбрать парамент в первом CheckBox. 
	Выберете определенную валюту, или можете посмотреть курсы всех валют, выбрав значение "Не выбрано".
5. Чтобы получить информация о курсах валют за определенный год, следует выбрать значение для второго CheckBox.
	Выберете определенный год, или можете посмотреть курсы валют за все года, выбрав значение "Не выбрано".
6. Чтобы получить информацию о курсе определенной валюты за определенный год, необходимо выбрать параметры для каждго выпадающего списка.


Построение графика выполтено с помощью JavaFX.

Все данные о курсах валют хранятся в файле cash.json.

Имеются данные о следующих валютах: евро, доллар, гривен, белорусский рубль, юань, франк, фунт.

Представленные значения валют - это средний курс за год.
